Preguntas Frecuentes

¿Cuál es la diferencia con Claude o ChatGPT?

AI Engine usa el modelo Mistral 7B, más pequeño pero altamente optimizado. Es gratuito, de código abierto y diseñado para deployment en producción.

¿Requiere GPU?

Funciona en CPU y GPU. Con GPU NVIDIA es 10-50x más rápido. La quantización 4-bit reduce memoria a 4GB en GPU.

¿Cómo funciona el Sistema RAG?

RAG busca documentos relevantes en tu base de conocimiento y los usa como contexto. Mejora precisión hasta 40% en respuestas específicas.

¿Puedo usarlo en producción?

Sí, está diseñado para producción. Incluye Docker, logging, health checks, validación robusta y API REST optimizada.
